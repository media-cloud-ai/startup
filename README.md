# Startup with Media Cloud AI

## Run the platform locally

To start the platform locally you've to install [docker](https://www.docker.com) & [docker-compose](https://docs.docker.com/compose/).
Once you get that simply to:

#### Backend startup

```bash
make backend-up
```

#### Workers startup

```bash
make backend-up
```

## Stop the platform

#### Backend

```bash
make backend-stop
```

#### Workers

```bash
make backend-stop
```
